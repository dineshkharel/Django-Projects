# Django-Projects
A collection of Django projects demonstrating various features, from CRUD operations to authentication and student management. Each project is self-contained and serves as a learning resource for different Django concepts.

# Projects in this repository
## 1. CRUD_operation_project

A simple Django project demonstrating Create, Read, Update, and Delete (CRUD) operations with a database.
Key Features:

Model-based data storage

Forms for creating and updating entries

List and detail views for displaying data

## 2. authentication_api

A Django project focused on user authentication using REST APIs.
Key Features:

User registration and login

Token-based authentication

Protected endpoints for authorized users

## 3. employee_auth_project

A Django project that combines employee management with authentication.
Key Features:

Employee CRUD operations

Role-based access control

Authentication and authorization for employees

## 4. todo_project

A classic To-Do application built with Django.
Key Features:

Add, update, delete, and mark tasks as complete

User authentication to manage personal tasks

Simple and clean UI for task management

## 5. Student_management_system

A Django project for managing students, courses, and grades.
Key Features:

CRUD operations for students, courses, and results

Search and filter functionality

Admin interface for easy management
